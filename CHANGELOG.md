# Changelog

## v1.2 — 28.06.2026

- Полный рефакторинг структуры модуля без смены публичного `id=ModuleConflictDetector` и имени `Module Conflict Detector`.
- Исправлен критический баг построения относительных путей, из-за которого реальные path-конфликты могли не определяться.
- CLI перенесён в самодостаточную директорию модуля: `bin/mcd-ctrl`.
- `/system/bin/mcd-ctrl` заменён на lightweight wrapper к модульному CLI.
- Добавлен `action.sh` для запуска сканирования из менеджера модулей.
- Добавлен `uninstall.sh` для очистки runtime-данных `/data/adb/mcd`.
- Добавлен lock от параллельных запусков `scan`.
- Добавлен конфиг `/data/adb/mcd/config.conf` с управлением автосканом после загрузки.
- Добавлена команда `mcd-ctrl doctor`.
- Whitelist переведён на безопасное точное сравнение без regex-побочных эффектов.
- Сканирование теперь учитывает обычные файлы, symlink и character-device/whiteout entries.
- Улучшено обнаружение `.replace`: одинаковые replace-каталоги и replace-каталог, перекрывающий дерево файлов другого модуля.
- Добавлено обнаружение потенциальных конфликтов ключей `system.prop`.
- Улучшена severity-классификация для `priv-app`, `app`, `framework`, `lib`, `permissions`, `sysconfig`, `init` и `sepolicy` путей.
- Обновлены README и usage-команды.

## v1.1 — 10.06.2026

- Добавлен whitelist конфликтов: `mcd-ctrl whitelist add|remove|list`.
- Добавлено обнаружение конфликтов `.replace`-каталогов.
- Исправлен счётчик конфликтов (неверный подсчёт из-за subshell).

## v1.0 — 09.06.2026

- Первый публичный релиз.
